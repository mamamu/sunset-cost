Ohio/Northern Kentucky Local History Combined Custom Image Search

This search engine uses Google Custom Search for the search API and indexes a few OH/KY local history photo databases provided by various libraries and historical organizations.

Searching for Image Data

Search in this format

https://sunset-cost.glitch.me/imagesearch/your search string/?offset=1

replacing the words your search string with your choice of search terms. Offset is optional and allows you to choose page number. You may also leave it blank for the first ten results

Returns JSON of Title, Snippet, Url, and image src for the custom search results. I am not returning alt text because many sites do not seem to provide it, thus the Title and snippet which appear to provide most of the information available about these photos.

This is not a really great search engine to look for Lolcats. However, if you want to get lots of results, searching 'flood' will do it. Other great searches: 'building','steamboat', and for a little fun 'zoo'.

Searching the Search History Data

You can also search like this to obtain data about the most recent searches performed

https://sunset-cost.glitch.me/recent/?offset=1

Offset is optional and allows you to choose results further in the set. You may also leave it blank for the first ten results
